# AI Crypto Guard - Chrome Extension

## Описание проекта
AI Crypto Guard - это расширение для Chrome, которое помогает пользователям защитить свои цифровые активы от криптовалютных скамов. Оно включает в себя различные функции для мониторинга и предотвращения несанкционированного доступа, а также автоматически сканирует веб-страницы на наличие криптовалютных адресов и проверяет их на безопасность.

## Возможности
- ✅ Проверка криптовалютных адресов Ethereum на безопасность
- ✅ Автоматическое сканирование веб-страниц на наличие адресов
- ✅ Выделение подозрительных адресов прямо на странице
- ✅ Оценка риска в процентах
- ✅ Статистика проверенных адресов
- ✅ Pre-Sign Guard для транзакций с обнаружением:
  - Неограниченных approve токенов
  - SetApprovalForAll для NFT
  - Подозрительных Permit подписей
  - Небольших ETH переводов (часто используются в скам-сценариях как "проверка кошелька")
  - Административных операций (transferOwnership, upgrade)
- ✅ Настраиваемый порог для определения "небольшого" ETH перевода

## Инструкции по установке
1. Клонируйте репозиторий на ваш локальный компьютер:
   ```bash
   git clone https://github.com/kaikakai016/ai-crypto-guard.git
   ```

2. Откройте Chrome и перейдите на страницу расширений:
   ```
   chrome://extensions/
   ```

3. Включите "Режим разработчика" (Developer mode) в правом верхнем углу

4. Нажмите кнопку "Загрузить распакованное расширение" (Load unpacked)

5. Выберите папку с проектом ai-crypto-guard

6. Расширение установлено! Значок появится в панели инструментов Chrome

## Руководство по использованию
1. Нажмите на иконку расширения в панели инструментов Chrome
2. Введите адрес криптовалютного кошелька в поле ввода
3. Нажмите "Проверить" для анализа адреса
4. Расширение автоматически сканирует открытые веб-страницы и выделяет подозрительные адреса красным цветом

### Настройки Guard
- **Enable Pre-Sign Guard**: Включает/отключает проверку транзакций перед подписью
- **Fail-Open**: При ошибках анализа разрешает операцию (безопаснее для пользователя)
- **Small Transfer Threshold**: Настройка порога для "небольших" ETH переводов (по умолчанию 0.001 ETH). Переводы ниже этого порога без данных будут считаться подозрительными.

### Обнаруженные угрозы
Расширение предупреждает о:
- Неограниченных approve токенов (MAX_UINT256)
- SetApprovalForAll для NFT коллекций
- Небольших ETH переводах без данных (типичная схема "проверки кошелька" в скамах)
- Подписях Permit (EIP-712)
- Административных операциях с смарт-контрактами
- Переводах больших сумм (>= 1 ETH)
